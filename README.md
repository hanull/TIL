## "Slow and steady wins the race"
- 학습한 내용, 알게된 내용 모든 것을 이해한 만큼 기록한다.
- 탄탄한 기본기를 갖추고, 변화하는 기술에 대한 지속적인 관심과 학습을 통해 배움을 멈추지 않는 것이 목표이다.
